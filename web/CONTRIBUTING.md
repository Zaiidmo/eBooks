# 🤝 Contributing to eBooks UI

Thanks for helping improve the **eBooks Frontend**!

## Workflow

1. **Fork** → **Branch** → **PR** to `main`
2. Branch names: `feat/*`, `fix/*`, `docs/*`, `chore/*`
3. Conventional commits: `feat:`, `fix:`, `docs:`, `refactor:`, etc.

## Setup

```bash
npm install
npm run dev
```

## Lint & Format

```bash
npm run lint
npm run format
```

## Testing (if enabled)

```bash
npm test
```

## PR Checklist

- [ ] Linked issue (if any)
- [ ] Screenshots for UI changes
- [ ] No secrets in diffs
- [ ] Updated docs if needed
